# Contributing to PromptLibrary

First of all, thank you for considering contributing to PromptLibrary.

Our goal is to build the world's most comprehensive open-source library of prompts, workflows, and prompt engineering resources. Every contribution helps improve the project for the entire community.

---

## Ways to Contribute

You can contribute by:

* Adding new prompts
* Improving existing prompts
* Creating workflows
* Writing guides and tutorials
* Fixing errors and typos
* Suggesting new categories
* Improving documentation

---

## Contribution Standards

Before submitting a contribution, please ensure that it:

* Provides clear value
* Is well-structured and easy to understand
* Uses proper formatting
* Avoids duplication
* Includes examples when applicable
* Follows repository conventions

---

## Prompt Submission Template

When adding a new prompt, use the following structure:

### Title

A short and descriptive title.

### Category

Example:

* Coding
* Research
* Writing
* Education
* Business
* Productivity
* Robotics
* Image Generation
* Video Generation

### Description

Explain what the prompt is designed to accomplish.

### Prompt

```text
Your prompt goes here.
```

### Expected Outcome

Describe the type of output users should expect.

### Tips

Optional recommendations for obtaining better results.

---

## Workflow Submission Template

### Workflow Name

Provide a descriptive workflow name.

### Goal

Explain the purpose of the workflow.

### Steps

1. Step One
2. Step Two
3. Step Three
4. Step Four

### Expected Result

Describe the final outcome.

---

## Pull Request Process

1. Fork the repository.
2. Create a feature branch.
3. Add your contribution.
4. Commit your changes.
5. Open a pull request.
6. Provide a clear description of your changes.

---

## Repository Structure

```text
prompts/
workflows/
resources/
benchmarks/
docs/
```

Please place your contribution in the appropriate directory.

---

## Community Guidelines

* Be respectful.
* Be constructive.
* Be collaborative.
* Help others learn.
* Focus on quality over quantity.
