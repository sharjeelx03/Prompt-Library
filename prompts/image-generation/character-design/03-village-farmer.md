# 🌾 Village Farmer

![Village Farmer](assets/village-farmer.png)

## Overview

A hardworking farmer whose family has cultivated the same land for generations. This example demonstrates realistic character design, cultural authenticity, environmental storytelling, and grounded visual narratives.
